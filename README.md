# Drama Mini Suite

TikTok Minis 短剧接入原型，包含移动端前台、CMS 管理台和本地模拟 API。

## Run

```bash
node server.js
```

- Mobile frontend: http://localhost:4173/
- CMS: http://localhost:4173/cms

## Scope

- 前台：短剧列表、分类/语言、播放页、剧集解锁状态、收藏、评论、用户钱包与观看记录。
- CMS：剧目/剧集管理、试看集数、解锁价格、分类/语言/地区、上传占位、评论审核、用户与充值消耗统计。
- API：本地 JSON 持久化，后续可替换为真实服务。
